Tomcat8.5.9启动流程分析2

catalina.sh 脚本

结束tomcat进程流程

脚本对tomcat进程的结束处理的比较优雅，先是通过PID判断进程是否存在，存在进程则先通过java程序内部来结束进程，如果停止失败再使用kill -15来停止进程，如果还结束进程失败，最后通过kill -9杀死进程

if判断 为ture的条件
-r FILE 文件存在且可读
-w FILE 文件存在且可写
-x FILE 文件存在且可执行
-z STRING 字符串为空
-n STRING 字符串非空
-f FILE 文件存在且是普通文件
-s FILE 文件存在且大小非0
-S FILE 文件存在且为套接字文件
-a FILE 文件存在
